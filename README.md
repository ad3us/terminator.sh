Bash script for detecting inactive, useless, zombie processess and services.

## Usage
Just download the file and make it executable:
```
$ chmod +x terminator.sh
```
and then run the script like this:
```
sudo ./terminator.sh
```
or like this:
```
sudo bash terminator.sh
```
